CSS selectors are patterns used to select and style HTML elements.

Syntax :

selector {
    property: value;
}
---

### ðŸ”¹ **Basic Selectors**
- `*` â€” Universal selector (selects all elements)
- `element` â€” Type selector (e.g., `div`, `p`, `h1`)
- `#id` â€” ID selector (e.g., `#header`)
- `.class` â€” Class selector (e.g., `.menu`)
- `[attribute]` â€” Attribute selector (e.g., `[type="text"]`)

---

### ðŸ”¹ **Combinator Selectors**
- `A B` â€” Descendant selector (selects `B` inside `A`)
- `A > B` â€” Child selector (selects direct children `B` of `A`)
- `A + B` â€” Adjacent sibling selector (selects `B` immediately after `A`)
- `A ~ B` â€” General sibling selector (selects all `B` siblings after `A`)

---

### ðŸ”¹ **Group Selector**
- `A, B` â€” Groups multiple selectors (e.g., `h1, h2, h3`)

---

### ðŸ”¹ **Attribute Selectors**
- `[attr]` â€” Elements with the attribute
- `[attr="value"]` â€” Exact match
- `[attr~="value"]` â€” Attribute contains word
- `[attr^="value"]` â€” Attribute starts with value
- `[attr$="value"]` â€” Attribute ends with value
- `[attr*="value"]` â€” Attribute contains substring

---

### ðŸ”¹ **Pseudo-classes**
- `:hover` â€” When mouse hovers
- `:active` â€” When element is active
- `:focus` â€” When element is focused
- `:visited` â€” Visited links
- `:first-child` â€” First child of parent
- `:last-child` â€” Last child of parent
- `:nth-child(n)` â€” nth child (e.g., `:nth-child(2)`)
- `:nth-of-type(n)` â€” nth of its type

---

### ðŸ”¹ **Pseudo-elements**
- `::before` â€” Insert content before element
- `::after` â€” Insert content after element
- `::first-letter` â€” Style first letter
- `::first-line` â€” Style first line
- `::selection` â€” Style selected text
- `::placeholder` â€” Style placeholder text

---

### ðŸ”¹ **UI Element States (CSS3)**
- `:valid`, `:invalid` â€” Form validation states
- `:in-range`, `:out-of-range` â€” Input range validation
- `:read-only`, `:read-write` â€” Input field states

---

### ðŸ”¹ **Logical Combinations (CSS4+)**
- `:is()` â€” Matches any of the selectors inside
- `:where()` â€” Like `:is()` but with no specificity
- `:has()` â€” Selects elements that contain a certain child (limited browser support)
